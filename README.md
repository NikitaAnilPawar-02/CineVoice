# CineVoice - Voice Enabled Movie Discovery Web App

CineVoice is a simple and fast React application that allows users to search items, save favorites, and perform searches using voice input. It is built with a clean UI and lightweight architecture.

## Features

- **Text Search:** Search any item instantly using a responsive input bar.
- **Voice Search:** Uses the Web Speech API to convert spoken words into text and perform searches.
- **Favorites System:** Save and manage your favorite items with one click.
- **Real-time Filtering:** Search results update instantly as the user types or speaks.
- **Clean and Minimal UI:** Focused on functionality first without heavy styling.

## Tech Stack

- React
- JavaScript
- HTML
- CSS
- React Icons
- Web Speech API for voice recognition
- Local Storage for saving favorites
